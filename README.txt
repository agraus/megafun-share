Учебный проект "Файлообменник"
Используются: PHP7.2.8, MySQL, Slim, getID3
Сервер: Apache 2.4
Функционал:
Загрузка файлов размером до 50мб на сервер.
Страница скачивания для каждого отдельного файла с описанием, комментарием и превью для изображений.
Страница с таблицей последних 100 загруженных файлов.
